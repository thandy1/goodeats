# GoodEats

A responsive multi-page demonstration e-commerce website for a concept food restaurant built with HTML5 and CSS3.

## Overview

GoodEats is a mockup restaurant website offering a variety of cuisines including burgers, pizza, tacos, chicken, seafood, and desserts. Built as a semester project for IMED 1341 Interface Design.

## Pages

- **Home** — Hero section, featured items, contact and hours of operation
- **Menu** — Full menu grid with 12 items across 6 cuisine types
- **Cart** — Static shopping cart with order summary
- **About** — Brand story, mission statement, and executive team
- **Privacy Policy** — Privacy policy and image citations

## Features

- Mobile-first responsive design with breakpoints at 768px and 1024px
- Pure CSS hamburger menu with clip-path animation — no JavaScript
- CSS custom properties for consistent color palette and typography scale
- Semantic HTML5 structure throughout
- Accessible images with descriptive alt text
- Responsive CSS Grid and Flexbox layouts

## Tech Stack

- HTML5
- CSS3
- Poppins — Google Fonts
- Phosphor Icons
